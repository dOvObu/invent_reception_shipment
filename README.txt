В папке с установленной программой находится файл _config.txt

Первая строка этого файла должна содержать
полный путь к xml файлу со списком номенклатур.


Пожалуйста, исправьте первую строку файла _config.txt

и сохраните изменения, нажав [ctrl]+[S]
